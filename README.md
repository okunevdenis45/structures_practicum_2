Инструкция внедрения базы данных:
•	Скачиваем СУБД MySQL 8.0.
•	Скачиваем любую СУБД для администрированием базой данных,к примеру Dbeaver.
•	В приложений 2,находится репозиторий базы данных с SQL-скриптами создания таблиц,индексов,функций,представлений,пользователей и полномочий,выбираем  и скачиваем SQL файл.
•	Открываем Dbeaver,нажимаем кнопку редактор SQL,выбираем кнопку новый редактор SQL , также нажимаем на кнопку редактор SQL и выбираем загрузить SQL скрипт и выбираем ранее скачанный sql файл
•	Когда вставили код в редактор, нажимаем комбинацию ctrl + enter и так со всеми файлами(Пример Dbeaver).
•	Готово!
